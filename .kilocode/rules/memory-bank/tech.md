# GridGuide - Technologies

## Framework & Language
- React 19 - Component-based UI framework
- TypeScript - Type-safe JavaScript superset
- Vite - Fast build tool and development server

## Styling & UI
- Tailwind CSS - Utility-first CSS framework
- shadcn/ui - Reusable UI components
- @fontsource-variable/inter - Variable font for Inter typeface
- tw-animate-css - Animation utilities

## PWA Implementation
- vite-plugin-pwa - PWA functionality plugin
- workbox-core, workbox-window - Service worker libraries
- PWA assets generation with pwa-assets-generator

## Analytics & Performance
- react-ga4 - Google Analytics 4 integration
- @microsoft/clarity - Microsoft Clarity analytics
- js-md5 - MD5 hashing utility

## Development Tools
- Biome - JavaScript/TypeScript linter and formatter
- Bun - JavaScript runtime
- Wrangler - Cloudflare deployment tool

## Build & Bundling
- Vite with React plugin for bundling
- Manual chunking for vendor and font assets
- Compression plugin for optimized assets

## Dependencies
- react, react-dom - Core React libraries
- react-router - Routing functionality
- react-helmet - Document head management
- @types/react - TypeScript definitions
- firebase - Firebase integration
- lucide-react - Icon library
- class-variance-authority - Class utility for variants
- clsx, tailwind-merge - Class name utilities
- framer-motion - Animation library
- autoprefixer, postcss - CSS processing
- tailwindcss-animate - Tailwind animation utilities
